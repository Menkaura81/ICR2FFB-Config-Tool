# ICR2FFB Config Tool
## A simple UI for the ICR2FFB plugin for Indicar Racing 2. 

<div align="center">
  <img src="https://github.com/user-attachments/assets/8ae95cf9-d67a-4500-b8e3-e187e8f21fac" alt="Imagen centrada" />
</div>

ICR2FFB Config Tool reads your ini file and show the values in the window. You can change them to your liking and then save the ini or launch the plugin.
To install it just copy ICR2FFConfig.exe in the same folder than the plugin
